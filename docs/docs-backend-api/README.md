# docs-backend-api

Dokumentasi backend API yang disusun dari file export Insomnia **HSE API MES**.

## Struktur

- `00-overview.md`
- `01-auth.md`
- `02-report.md`

## Catatan penting

- Dokumen ini disusun dari koleksi request Insomnia, bukan dari source code backend atau OpenAPI spec.
- Contoh kredensial, token, dan path yang sensitif telah **disamarkan**.
- Beberapa format respons belum dapat dipastikan karena tidak tersedia contoh respons di file sumber.
- Untuk endpoint follow-up, path ditulis sesuai koleksi sumber dan perlu diverifikasi langsung ke backend bila implementasi final berbeda.
